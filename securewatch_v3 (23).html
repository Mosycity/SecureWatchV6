📊 How the Threat Score is Calculated
Each CVE is scored from 0–100+ based on intelligence signals. Higher score = higher urgency tier.
SignalPointsWhat it means
⚡ KEV+50CISA confirmed: actively exploited in the wild right now
💣 PoC Public+30Proof-of-concept exploit code is publicly available — any attacker can use it
EPSS ≥ 50%+30ML model predicts very high probability of exploitation in next 30 days
EPSS ≥ 20%+18Elevated exploitation probability — top percentile of all CVEs
EPSS ≥ 10%+8Moderate exploitation probability worth monitoring

🆕 <24h old+20Published within 24 hours — attackers are also just seeing this now
CVSS ≥ 9.5+20Critical severity — maximum impact if exploited
🌐 Network AV+15Remotely exploitable over the network without physical/local access
CVSS ≥ 8.0+12High severity with significant potential impact

<7 days old+6Recent enough to still be in active attacker targeting window
